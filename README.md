
ownip
===========


ownip is a thin wrapper on top of the [MYIP-API](https://api.myip.com). 


To Start...
-----------

To install ownip:

```bash
pip install ownip
```


Using it...
-----------

```python
from ownip import OwnIp

ip_dict = OwnIp.get_ip()
print(ip_dict)
```
